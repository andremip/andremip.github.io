    Projects

    1 vs 1 Survival Game

    1 vs 1 videogame developed using Phaser 3 where an exorcist flees from a demon while collection candles to complete a ritual. The game was developed as part of a team project for university, with client-server communication via API REST and Websockets (not active on itch.io).

    No Sense - Game Jam Game Scholars

    Videogame I developed with some friends for the Game Scholars' 2023 Game Jam, where we won 3 rd place. In this game, developed in Unity, I was responsible for part of the programming and the animation of some non-character elements.

    Deformable solid affected by wind

    Simulation in Unity of a 3D deformable solid using the mass-spring method, with wind affecting only the faces influenced by the current wind direction. The video also includes a simulation of waving flags using the same method.

    Missile Command

    Arcade game inspired by the game "Missile Command," programmed in JavaScript. Click the screen to shoot missiles from your bases and destroy enemy attacks. Manage your ammo and protect your cities as you advance through levels.
